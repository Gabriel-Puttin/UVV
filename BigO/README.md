## Executando aplicação (LOCALMENTE)

1. INICIALIZE O AMBIENTE VIRTUAL DO PYTHON
```
python -m venv .venv
```
2. ATIVE O AMBIENTE VIRTUAL DO PYTHON
```
\.venv\Scripts\Activate.ps1
```
3. INSTALE AS DEPEDÊNCIAS DO PROJETO
```
pip install -r .\requirements.txt
```
4. RODE O ARQUIVO eda.py
```
py eda.py
```
